# Prompts

This folder is dedicated to showcasing a variety of AI prompt templates and experiments. Prompts here are designed for generating images, videos, or transforming media in creative ways.

## What's Inside

- **Text-to-Image Prompts**: Written descriptions that produce unique images using generative models.
- **Text-to-Video Prompts**: Ideas or scripts that result in custom animated clips or video sequences.
- **Image-to-Image Prompts**: Instructions for transforming or editing an existing image with AI.
- **Image-to-Video Prompts**: Guidance for turning a static image into animated visuals or motion.
- **Reference-to-Video Prompts**: Prompts that blend reference images or styles into new video content.

## How to Use

Browse the prompt files to find inspiration for your own projects. Each prompt typically comes with:

- A sample input
- Output examples (image/video previews)
- Tips on getting the best results with various models
