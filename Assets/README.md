# Assets

A library of finished media created by AI, available for review, inspiration, or further testing.

## What's Inside

- **Sample Images**: Galleries of various styles, moods, and subject matter generated from prompts.
- **Sample Videos**: Clips showcasing animations, transitions, and narrative sequences produced by generative models.

## How to Use

Browse through the samples to see what's possible, use them for benchmarking your own outputs, or remix them for personal projects. Each asset may include the original prompt, model info, and notes on generation parameters.
