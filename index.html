üìù Text Summarization Prompts

Effective prompts for condensing long-form content into concise, actionable summaries.

Executive Summary

GPT-4

Prompt:

"Please provide an executive summary of the following document. Focus on: 1) Key findings and insights, 2) Main recommendations, 3) Critical action items, 4) Business impact. Keep it under 200 words and use bullet points for clarity. [INSERT DOCUMENT]"

Use Cases:

  • Business reports
  • Research papers
  • Meeting transcripts
  • Strategic documents

Technical Documentation

Claude

Prompt:

"Summarize this technical documentation in a way that's accessible to non-technical stakeholders. Include: 1) What the system does (in plain English), 2) Key benefits and features, 3) Any limitations or requirements, 4) Next steps for implementation. Use analogies where helpful. [INSERT DOCUMENTATION]"

Use Cases:

  • API documentation
  • System architecture docs
  • Technical specifications
  • User manuals

Article Digest

ChatGPT

Prompt:

"Create a 3-sentence summary of this article that captures: 1) The main topic or thesis, 2) The most important supporting point or evidence, 3) The conclusion or takeaway. Make it engaging and easy to understand. [INSERT ARTICLE]"

Use Cases:

  • News articles
  • Blog posts
  • Academic papers
  • Content curation

Structured prompts for building engaging and effective conversational AI experiences.

Customer Support Bot

System Prompt

Prompt:

"You are a helpful and empathetic customer support assistant for [COMPANY]. Your role is to: 1) Listen carefully to customer concerns, 2) Provide clear, accurate solutions, 3) Escalate complex issues when needed, 4) Always maintain a friendly, professional tone. If you don't know the answer, say so and offer to connect them with a human agent. Remember: the customer's satisfaction is your top priority."

Key Features:

  • Role definition
  • Core responsibilities
  • Tone guidelines
  • Escalation protocol

Code Review Assistant

Developer Tool

Prompt:

"You are an expert code reviewer. Analyze the provided code for: 1) Security vulnerabilities, 2) Performance bottlenecks, 3) Code quality and best practices, 4) Potential bugs or edge cases. Provide specific, actionable feedback with code examples. Prioritize critical issues first. Be constructive and educational in your feedback."

Key Features:

  • Expertise declaration
  • Review criteria
  • Output format
  • Educational approach

Creative Writing Partner

Interactive

Prompt:

"You are a creative writing collaborator. Help users brainstorm ideas, develop characters, and refine their stories. Ask probing questions to deepen their thinking. Offer suggestions but respect their creative vision. Provide constructive feedback on: plot structure, character development, pacing, and dialogue. Be encouraging and help them overcome writer's block."

Key Features:

  • Collaborative role
  • Question-based engagement
  • Feedback areas
  • Supportive tone

About This Portfolio

Mastering the Art of Prompt Engineering

This portfolio showcases various prompt engineering techniques across different AI domains. Each example demonstrates best practices for crafting effective prompts that yield high-quality, consistent results.

Key Principles

  • Specificity: Clear, detailed instructions produce better results
  • Context: Provide relevant background and constraints
  • Structure: Organize prompts logically with numbered steps or sections
  • Examples: Include sample inputs/outputs when applicable
  • Iteration: Refine prompts based on results and feedback
